## Viral Mechanics for Citadel AI

### K-Factor Formula
```
K = i × c
i = invitations sent per user
c = conversion rate of invitations
K > 1 = viral (each user brings in more than 1 new user)
```

### Growth Loops (not funnels — loops compound)

**Content → Affiliate Loop (Stream 1 + 3)**
```
Viral video → viewer clicks affiliate link → purchase
→ positive review → video ranks higher → more viewers → repeat
```
Optimize: A/B test affiliate link placement, product-to-niche relevance

**Agency Lead Loop (Stream 2)**
```
Signal-based cold email (12-25% reply) → discovery call (SPIN)
→ signed client → published case study
→ case study in next 10 emails → higher reply rate → more clients
```

## Experimentation Discipline

### Experiment Design
```
Hypothesis: If we [change X], then [metric Y] improves by [Z%] because [reason]
Control: [current version]
Variant: [proposed change]
Sample size: minimum 15 per variant (95% confidence)
Success metric: [specific, measurable]
```

### ICE Prioritization
- **I**mpact × **C**onfidence × **E**ase — run highest-score experiments first
- Change one variable at a time — if you change everything, you learn nothing
- Never call a winner before 95% statistical significance
