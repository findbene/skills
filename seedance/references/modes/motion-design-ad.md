# Mode: motion-design-ad — Motion Design Advertisement (SaaS/Tech)

Generate motion design advertisement prompts for software and tech companies: UI morphing, data visualization, device reveals, code-to-product transformation.

## Process

1. Collect context: product type (SaaS, mobile app, dev tool, analytics platform), key benefit to visualize, duration (6–30s for ads), platform ratio
2. Read `~/.claude/skills/seedance-motion-design-ad/references/details.md` for full hook library, UI motion vocabulary, and ad structure templates
3. Apply 2-second tech hook: data visualization explosion, UI morphing, device appearing from void, code-to-product transformation, loading bar to checkmark, particle system assembly
4. Specify motion design vocabulary: easing curves (ease-in, ease-out, spring), element transitions (scale, fade, slide, morph), timing precision
5. Describe visual hierarchy: what element dominates frame at each moment, reading order for multi-element scenes
6. Include UI/product representation: dashboard elements, interface components, data charts — describe their animation behavior
7. Specify typography animation if text is part of the ad: typewriter, slide-in, scale-up
8. Define color palette: brand-appropriate tech aesthetic (dark mode, light mode, gradient choices)
9. Use 60fps for fluid tech animations (UI interactions feel smoother)
10. Build complete prompt: hook + UI motion + hierarchy + typography + color + timing
11. Verify: product benefit is visually communicated, not just described; no live-action mixed with motion design terms

## Notes
- Timing precision is critical for tech ads — specify milliseconds for key moments
